# 紫微斗数角色设计工具 - 安装使用说明

## 支持平台

- ✅ macOS (Mac电脑)
- ✅ Windows (Windows电脑)
- ✅ iOS (iPhone/iPad)
- ✅ Android (安卓手机/平板)

---

## 一、电脑端使用（macOS/Windows）

### 方法1：直接打开（最简单）

1. 解压下载的压缩包
2. 双击 `index.html` 文件
3. 系统会自动用默认浏览器打开
4. 即可开始使用

### 方法2：安装为桌面应用（推荐）

#### macOS:

1. 用 Chrome 或 Edge 浏览器打开 `index.html`
2. 点击浏览器地址栏右侧的 **安装图标**（➕ 或 📲）
3. 选择"安装"
4. 安装完成后，会在启动台/应用程序中创建图标
5. 以后可以直接从启动台打开，无需浏览器

#### Windows:

1. 用 Chrome 或 Edge 浏览器打开 `index.html`
2. 点击右上角菜单（⋮）
3. 选择"更多工具" → "创建快捷方式"
4. 勾选"作为窗口打开"
5. 点击"创建"
6. 桌面会生成应用图标，双击即可使用

---

## 二、iOS 使用（iPhone/iPad）

### 安装到主屏幕

1. 用 Safari 浏览器打开工具（可以通过AirDrop或iCloud Drive传输文件后用Safari打开）
2. 点击底部分享按钮（⬆️）
3. 向上滑动，选择"添加到主屏幕"
4. 可以修改名称，然后点击"添加"
5. 主屏幕会出现应用图标，像普通App一样使用

### 离线使用

- 首次打开后，工具会自动缓存
- 之后无需网络也能使用
- 可以随时随地创作角色

---

## 三、Android 使用

### 安装到主屏幕

1. 用 Chrome 浏览器打开工具
2. 点击右上角菜单（⋮）
3. 选择"添加到主屏幕"或"安装应用"
4. 确认添加
5. 主屏幕会出现应用图标

### 离线使用

- 首次打开后自动缓存
- 支持离线使用

---

## 四、文件传输方法

### Mac → iPhone/iPad

**方法1：AirDrop**
1. 在Mac上右键点击文件夹
2. 选择"共享" → "AirDrop"
3. 选择你的iPhone/iPad
4. 在设备上接收文件

**方法2：iCloud Drive**
1. 将文件夹复制到 iCloud Drive
2. 在iPhone/iPad上打开"文件"App
3. 找到文件夹，用Safari打开 index.html

**方法3：微信/QQ**
1. 将文件夹压缩成zip
2. 发送到微信/QQ文件传输助手
3. 在手机上解压并打开

### Windows → 手机

**方法1：微信/QQ文件传输**
1. 压缩文件夹
2. 发送到微信/QQ
3. 手机接收并解压

**方法2：数据线传输**
1. 用数据线连接手机和电脑
2. 复制文件夹到手机存储
3. 用手机浏览器打开

**方法3：网盘**
1. 上传到百度网盘/阿里云盘等
2. 手机下载并打开

## 五、使用技巧

### 快捷操作

- **复制结果**：生成人物设定后，点击"复制结果"按钮，可直接粘贴到剧本软件
- **重新匹配**：对命盘不满意可以返回重新选择属性
- **离线使用**：首次加载后，关闭网络也能正常使用

### 数据保存

- 工具不会上传任何数据到服务器
- 所有数据都在本地处理
- 可以放心使用，保护创作隐私

### 推荐浏览器

- **电脑**：Chrome、Edge、Safari
- **iOS**：Safari（推荐，支持添加到主屏幕）
- **Android**：Chrome、Samsung Internet

## 六、常见问题

### Q: 为什么无法安装到主屏幕？
A: 请确保使用 Safari（iOS）或 Chrome（Android）浏览器，其他浏览器可能不支持PWA安装。

### Q: 离线使用时某些功能失效？
A: 首次打开时请保持网络连接，让Service Worker完成缓存。之后即可完全离线使用。

### Q: 如何更新到新版？
A: 重新下载最新版本的压缩包，替换旧文件即可。浏览器会自动更新缓存。

### Q: iOS上如何解压文件？
A: iOS自带解压功能，点击zip文件即可自动解压。或使用"文件"App中的解压功能。

## 七、技术说明

本工具采用 PWA（Progressive Web App）技术：
- 支持离线使用（Service Worker缓存）
- 可安装到主屏幕（像原生App一样使用）
- 响应式设计（适配各种屏幕尺寸）
- 本地运行（数据不上传服务器）
